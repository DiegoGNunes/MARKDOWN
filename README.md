## Condições
> if, switch, ternarios

Condições é o *método* para **validar** se uma ***informação*** é verdadeira, caso seja verdadeira a condição aplicada `if`, executa somente o código que está dentro dele, com isso temos condições para que o código execute somente o necessário.

```csharp
if (valor1 == valor2)
{
    MessageBox.Show("Os números são iguais")
}
```
